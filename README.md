- Java 8
- Spring Boot
- Spring MVC (REST)
- Spring Data JPA
- MySQL (XAMPP)
- Lombok
- Maven
- ModelMapper

# Frontend technologies
- IntelliJ IDEA
- Typescipt
- Angular (Angular CLI)
- NPM
- Sass (scss)
- Angular Material Design
- ng-http-loader
